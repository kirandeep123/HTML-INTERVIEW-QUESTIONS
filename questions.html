Key Differences between local storage , session storage objects?
Local storage:
  1. No expiry date
  2. data will not be deleted when browser window closes.
  3. data is accessible on multiple window on the browser
 Session storage :
  1. stored only per session
  2. data will be lost once browser window closes
  3. data is accessible only on the current window of the browser.
